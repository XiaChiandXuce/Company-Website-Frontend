# TYH Company - 定制化软件开发官方网站

## 项目简介
这是 TYH Company 的官方网站项目，旨在展示我们高端的定制化软件开发服务。本项目采用现代化的前端技术栈，致力于打造高性能、视觉独特且用户体验一流的 Web 应用。

## 技术栈
本项目基于以下核心技术构建：
-   **构建工具**: [Vite](https://vitejs.dev/) - 极速的开发服务器和构建工具
-   **框架**: [React](https://react.dev/) - 用于构建用户界面的 JavaScript 库
-   **语言**: [TypeScript](https://www.typescriptlang.org/) - 提供强类型支持
-   **设计系统**: Vanilla CSS + CSS Variables (定制化高端设计，不依赖 UI 框架)
-   **路由**: React Router v6
-   **动画**: Framer Motion

## 开发指南

### 环境准备
确保您的环境中已安装 [Node.js](https://nodejs.org/) (推荐 v18+)。

### 安装依赖
```bash
npm install
```

### 启动开发服务器
```bash
npm run dev
```
访问 http://localhost:5173 查看效果。

### 构建生产版本
```bash
npm run build
```

## 设计理念
本项目遵循 **"Distinctive & Premium"** (独特与高端) 的设计原则，拒绝千篇一律的 AI 生成风格。我们将注重微交互、排版不仅仅是字体大小、以及流畅的空间布局。
